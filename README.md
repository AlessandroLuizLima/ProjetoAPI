# DocMind 🧠📄

DocMind é uma aplicação desktop desenvolvida em Python que utiliza a API do Claude (Anthropic) para auxiliar usuários na leitura e compreensão de documentos PDF.

O sistema permite importar qualquer arquivo PDF, gerar automaticamente um resumo do conteúdo em português e fazer perguntas específicas sobre o documento — recebendo respostas baseadas exclusivamente no que está escrito no arquivo. Todas as interações ficam salvas em um banco de dados PostgreSQL, criando um histórico consultável de documentos analisados, resumos gerados e perguntas respondidas.

A interface é construída com Tkinter, tornando o uso simples e acessível para qualquer perfil de usuário, sem necessidade de conhecimento técnico. O sistema também conta com leitura em voz alta do resumo gerado, utilizando a biblioteca gTTS.

O projeto foi desenvolvido como trabalho acadêmico da disciplina de Programação com Inteligência Artificial, curso de Análise e Desenvolvimento de Sistemas.
